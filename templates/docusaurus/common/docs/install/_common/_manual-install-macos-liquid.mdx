### Download

The macOS versions of **xPack {{ appName }}**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-{{ appLcName }}-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-x64.tar.gz</code>
* <code>xpack-{{ appLcName }}-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack {{ appName }}</b>,
unpack the archive and move it to <code>~/Library/xPacks/{{ appLcName }}</code> folder,
for example <code>/Users/ilg/Library/xPacks/{{ appLcName }}/xpack-{{ appLcName }}-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/Library/xPacks/{{ appLcName }}
cd ~/Library/xPacks/{{ appLcName }}

tar xvf ~/Downloads/xpack-{{ appLcName }}-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }-darwin-x64.tar.gz
chmod -R -w xpack-{{ appLcName }}-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }`
